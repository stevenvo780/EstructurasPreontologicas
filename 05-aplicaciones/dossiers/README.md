# Dossiers de anclaje detallados

Esta subcarpeta aloja dossiers de anclaje completos por aplicación cuando se eleven a modo demostrativo. Cada dossier sigue la plantilla del capítulo `03-formalizacion/03-auditoria-ontologica-y-diseno-de-investigacion.md` §9.

## Estado actual

Vacío. El caso ancla canónico (capítulo `05-dinamica-conductual-reconstruccion-warren.md`) tiene su dossier integrado en el capítulo principal.

## Cuándo crear archivos aquí

- cuando una aplicación programática (mente, biología, sistemas técnicos, instituciones) avance hacia modo demostrativo;
- cuando un dossier supere ~5 páginas y convenga sacarlo del capítulo principal;
- cuando se construyan dossiers comparativos para benchmarks (mega-tarea 5).

## Convención de nombres

`[NN]-[dominio]-[fecha].md`, donde NN es el número del capítulo de aplicación correspondiente.
