# Protocolo de simulación — 24_caso_microplasticos

## Sonda de Jambeck con difusión oceánica

**Régimen físico:** Flujo de masa continental-oceánico.

## Ecuación de la sonda

```
dM/dt = entrada_costera*población - degradación + transporte_corrientes
```

Modelo de input continental Jambeck (2015).

## Origen de parámetros

OWID + Jambeck data.
